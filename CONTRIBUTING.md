# Contributing to RakshaSOS

Thank you for considering contributing to RakshaSOS.

## How to Contribute

1 Fork the repository

2 Clone your fork

git clone https://github.com/varun5041/RakshaSOS.git

3 Create a branch

git checkout -b feature-name

4 Make your changes

5 Commit changes

git commit -m "Added new feature"

6 Push

git push origin feature-name

7 Create a Pull Request

## Guidelines

• Write clean and readable code
• Follow Android development standards
• Test before submitting pull request
